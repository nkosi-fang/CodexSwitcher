## 变更
修复了新增账号后 base_url 写错的问题，现在会按当前账号正确替换。
新增账号后会自动选中新账号，并可一键应用。
一键启动 VS Code 时不再弹命令行黑窗。
中转站接口检测不再只看 200，会校验返回内容是否有效。
软件运行时会自动检查更新，不用手动点“立即检查”。
左侧“检查更新”新增红点数字提醒，按版本差显示数量。
会话右键新增“继续该会话（VS Code）”，并支持直达会话。
会话右键新增“删除该会话”，可快速清理无效会话。
会话页提示文案和“统一清理”标题已按新样式调整。
开发者反馈区改为二维码反馈，并隐藏二维码本地路径显示。
