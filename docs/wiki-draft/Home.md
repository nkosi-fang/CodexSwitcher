## 1. 文档定位

CodexSwitcher 是一个面向 Windows 的桌面工具，核心目标是把 Codex CLI、VSCode Codex 插件、账号配置与会话管理整合到同一个可视化入口中，降低手工改配置、手工排障的成本。

本 Wiki 重点回答三类问题：

- 这个工具能做什么（功能边界）
- 每个功能如何工作（技术原理）
- 如何稳定发布与维护（构建、校验、排障）

## 2. 系统视角总览

从代码结构看，系统分为两层：

- `pyside_switcher.py`：UI 页面、交互编排、后台任务调度
- `codex_switcher.py`：共享能力（账号存储、配置写入、网络探测、命令发现）

运行时通过 `AppState` 管理跨页面共享状态（当前账号、codex 路径、VS Code 安装目录等），并通过后台线程 + `run_in_ui(...)` 保证“耗时任务不阻塞界面”。
